Accomplishments & Reflections

So far, we have the project setup in Unity using compute shaders, with rendering complete using a material shader. Alongside this, we also have some basic mouse interactions to remove obstructing terrain.

We are currently working on implementing the Eulerian fluid simulation, with an approximate time of finishing by the weekend of April 19. We plan for this to be a stepping stone for FLIP fluid simulation, which builds on top of Eulerian to add particle simulations. The particle nature of FLIP will come in handy for discretely counting fluid flow for our game.

At the same time, we also have a couple level designs setup for the game (we are going for a Where's My Water clone). We're also in the middle of implementing the level loader, which loads levels from a texture file that describes the level.

We're making good pace so far towards our goal. Despite accidentally thinking we had one more week than we actually did, we should still be on track to finishing our project by the deadline with some time to spare.
