Voxel Falling Sand with Ray Marching Smoothing

Team: Ann Basil, David Hernandez, Ben Bliss, Jinkang Fang

A 3D falling sand game with kinematics and chemical reactions.
Implemented with voxel grids and ray marching smoothing.

Problem Description

Falling sand games are a great tool to demonstrate how elements like sand and water interact with each other. However, traditional falling sand games are made in 2D, making them more of a fun game to think about but that's it. We want to expand on the concept of falling sand games into 3D.

To improve realism, each voxel should be small, just like traditional falling sand games. We also aim to use ray marching with a controllable delimiter to trace a boundary around the corners of each voxel. The idea is to create a more dynamic experience with smoothing, but in a controlled manner, such that it doesn't appear as liquid.
