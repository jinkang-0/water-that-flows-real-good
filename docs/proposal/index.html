Interactive Fluid Simulation

Team: Ann Basil, Ben Bliss, David Hernandez, Jinkang Fang

Summary
A real-time, interactive 2D fluid simulation with obstacles, mouse interaction, fluid interactions, and minigames. Play with it in real-time on the browser.

Webpage: jinkang-0.github.io/water-that-flows-real-good/proposal/

Problem Description

We were inspired by the mobile game Where's My Water and wanted to recreate it with different fluid types that can be interacted with on the web.

We believe this is an important problem because it gives us a sense of how fluid mechanics can be used to enhance realism in video games. However, it is also a tough problem to tackle as fluid mechanics involve lots of math and physics.

Fortunately, there are lots of great resources online to learn about existing models of fluid mechanics. We will spend some time studying different fluid simulation models to settle on a realistic particle-like simulation that can be placed in a video game. We can then implement it in Unity and extend it into a game. If time allows, we can also apply shaders to add white splashes, making the water more realistic.

Goals and Deliverables

For our final deliverable, we want to recreate a Whereâ€™s My Water clone using 2D fluid simulations, complete with obstacles, mouse interactions, fluid interactions, and scoring. This will be available as an interactive demo on the web.

To measure the quality of our system, we will use FPS as a key metric. We want the simulation to run on smooth frame rates (ideally 60fps) on the web, using a modern laptop. Some question to answer: is it possible to use 2D fluid simulations and still achieve smooth frame rates in a video game? What other applications can we use fluid simulations in?

We believe having a 2D fluid simulation model with some interactivity will give us an A on the project. However, we aim to pair this with a complete game with realistic water rendering and different fluid interactions.
